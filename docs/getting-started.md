# ▶️ Getting Started

{% hint style="info" %}
Python3 or above is required
{% endhint %}

### Setup

#### **Clone the repository**

```bash
git clone https://github.com/p1ngul1n0/blackbird
cd blackbird
```

#### **Install requirements**

```bash
pip install -r requirements.txt
```

#### **✨ You're good to go!**
